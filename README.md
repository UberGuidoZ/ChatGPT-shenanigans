# ChatGPT-shenanigans
Well... ChatGPT Shenanigans!

## AI Bingo Card Generator

Generate bingo cards filled with common ChatGPT-isms and play interactively in your terminal. Spot the AI cliches in real time!

### Quick Start

```bash
python -m ai_bingo
```

Requires Python 3.6+ with no external dependencies.

### Features

- **Random card generation** — Every card is unique
- **Seeded cards** — Use a seed to share identical cards with friends
- **Interactive play mode** — Mark squares as you spot AI-isms, auto-detects BINGO
- **80 phrases across 8 categories** — Disclaimers, enthusiasm, hedging, filler words, hallucinations, and more
- **Export to text file** — Save cards for printing or sharing
- **Browse mode** — Explore the full phrase database by category

### Phrase Categories

| Category | Examples |
|----------|---------|
| Disclaimers | "As an AI language model...", "I can't browse the internet..." |
| Enthusiasm | "Great question!", "I'd be happy to help!" |
| Structure | "Let me break this down...", "Here are some key points:" |
| Hedging | "It depends on the context...", "This is a nuanced topic..." |
| Filler | "Delve into...", "Robust solution", "Tapestry of..." |
| Closers | "Is there anything else I can help with?", "I hope this helps!" |
| Behaviors | "Refuses then does it anyway", "Makes up a citation" |
| Hallucinations | "Cites a paper that doesn't exist", "Invents a plausible-sounding URL" |
