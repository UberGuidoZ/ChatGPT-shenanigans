# AI Hallucination Quiz

Can you tell the difference between real facts and AI hallucinations? Test your ability to spot confident nonsense.

## Quick Start

```bash
python -m ai_hallucination_quiz
```

Requires Python 3.6+ with no external dependencies.

## Game Modes

| Mode | Description |
|------|-------------|
| Quick Game | 10 random questions |
| Full Game | All 40 questions |
| Category Challenge | Pick a specific topic |
| Custom Game | Choose your question count |
| Browse | Explore all facts with explanations |

## Categories

- **History** — Napoleon's rabbits, Viking helmets, and more
- **Science** — Neutron stars, octopus hearts, and persistent myths
- **Technology** — The first bug, the first webcam, AI origins
- **Geography & Nature** — Pink lakes, berry botany, goldfish memory
- **Literature & Culture** — Dr. Seuss bets, fortune cookie origins
- **AI & Computing** — Hallucination meta-facts, AI milestones

## Scoring

| Grade | Score |
|-------|-------|
| S | 100% — Hallucination-proof |
| A | 90%+ — Excellent detector |
| B | 80%+ — Strong BS radar |
| C | 70%+ — Some slipped through |
| D | 60%+ — Needs calibration |
| F | <60% — The AI wins this round |

Streak tracking rewards consecutive correct answers.
