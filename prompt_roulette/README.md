# Prompt Roulette

Spin the wheel and get a wild, absurd, or surprisingly deep AI prompt. Mad libs meets prompt engineering — over 36 million possible combinations.

## Quick Start

```bash
python -m prompt_roulette
```

Requires Python 3.6+ with no external dependencies.

## Features

- **Spin** — Generate a random prompt instantly
- **Complexity levels** — From simple 2-slot combos to 5-slot maximum chaos
- **Reroll** — Don't like one part? Reroll individual slots without changing the rest
- **Batch mode** — Generate up to 20 unique prompts at once with optional export
- **Workshop** — Hand-pick each slot or mix manual + random for full control
- **110 phrases** across 6 slot categories

## Slot Categories

| Slot | Examples |
|------|---------|
| Role | "a sentient rubber duck", "a ghost haunting an IKEA" |
| Task | "explain quantum physics", "write a breakup letter" |
| Style | "in Shakespearean English", "like a true crime podcast" |
| Constraint | "but every sentence must include a fruit", "while getting progressively sleepier" |
| Audience | "for an audience of confused cats", "to a jury deciding if pineapple belongs on pizza" |
| Setting | "on a malfunctioning space station", "during a potluck where everyone brought potato salad" |

## Complexity Levels

| Level | Slots | Vibe |
|-------|-------|------|
| * | 2 | Quick and fun |
| ** | 3 | A bit wilder |
| *** | 4 | Getting absurd |
| **** | 5 | Maximum nonsense |

## Sample Outputs

> You are a motivational speaker for houseplants, on the last day the internet exists. Your task: narrate a nature documentary, using only food metaphors, while making increasingly bold predictions about the future.

> You are a conspiracy theorist who is actually right about everything, on a malfunctioning space station. Please write a breakup letter, as if explaining it to a very smart dog.

> Please explain the meaning of life, as a country western song.
